# Лабораторная работа: Анализ журнала событий (Log Analysis)

## Цель работы
Научиться осмысленно читать журнал событий, восстанавливать последовательность происходившего, отличать симптом от первопричины, формулировать гипотезу о сбое и предлагать меры по предотвращению повторения инцидента.
